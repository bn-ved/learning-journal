and katas across languages live here. When something grows into a real project,
it **graduates to its own repo** and gets pinned on my profile.

## Journal
Per-session log under [`journal/`](./journal/) — newest entries at the top of each
month file. Each entry: **what I did / what clicked / what's next**.

## Commit convention
| Prefix | Use |
|--------|-----|
| `learn:` | new concept or exercise |
| `solve:` | a problem / kata solved |
| `feat:` / `fix:` | real project work |
| `wip:` | unfinished checkpoint |

Example: `learn(go): worker pool with goroutines + channels`

## Workflow
1. **Per session** → one focused commit + one entry in the current month's journal file.
2. **End of week** → skim the journal, tick roadmap boxes, jot a 1-line retro.
